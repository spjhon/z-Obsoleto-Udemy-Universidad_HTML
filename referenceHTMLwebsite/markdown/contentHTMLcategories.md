HTML <blockquote> Tag
<!--titulo-->
<!--parrafo-->
A section that is quoted from another source. **Tip:** Use `<q>` for inline (short) quotations.  

**Default CSS Settings:**  
```css
blockquote {
  display: block;
  margin-top: 1em;
  margin-bottom: 1em;
  margin-left: 40px;
  margin-right: 40px;}
```
